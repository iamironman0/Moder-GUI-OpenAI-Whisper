## Setup

### Requirements
* Python version 3.9
* Torch with Cuda
* Nvidia GPU
* ffmpeg
* customtkinter 

> For more information please visit OpenAI Wishper github: https://github.com/openai/whisper

1. Run this command
```
pip install -r requirements.txt
```
2. Run the file

```
python main.py 
```
